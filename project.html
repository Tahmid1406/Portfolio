Project Title : Human Activity Recognition Using Machine Learning Algorithms.

Instructor : Syed Athar Bin Amir | NSU Profile | Google Scholar

Course : CSE445 - Machine Learning

Human activity recognition can be proven beneficial to several sectors including human-machine interaction, building smart devices with exclusive features and understanding human nature. Smartphones are one of the smart gadgets that stays close to a person for the better part of a day. Smartphones have several sensors which are capable of reading angular motion, acceleration, geographic location and tri-axial (3-d) motion data very accurately. Machine learning techniques have already proven their capability to work with data in large scale and having large number of feature. In this project we used a sensor read value dataset to classify human activity based on the data read by their cellphone sensors. The activity has been classified for 6 classes and a total of 3 machine learning models have been tested on this dataset. SVC performes with a highest testing accuracy of 95% followed by logistic regression with testing accuracy of 82% followed by Ridge classifier CV with testing accuracy of 81%. For further investigation about the models performance, we showed the precision, recall, f1-score along with corresponding confusion matrix.

WELink | Part-time Job Finder Android Application.

Instructor : Dr. Mohammad Abu Yousuf | Google Scholar

Course : CSE215 - Programming Language II (Introduction to Java)

WELink is a android software project where users (directed towards students) can find part-time jobs and people can post if they have any part-time jobs. In bangladesh, part-time student jobs is still not seen as a respectable thing to do. The purpose of this project is to provide a platform where stream of part-time jobs will be available and we hope people will get accustommed with our app providing a platform. In our app, user can post, comment and upload cv.
